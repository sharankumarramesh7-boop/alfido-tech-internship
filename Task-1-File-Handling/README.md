# 📁 Task 1 – Python File Handling & Automation

## 📌 Overview

This project demonstrates the fundamentals of **Python File Handling**, **File Automation**, and **Exception Handling**. It showcases how Python can be used to perform common file operations efficiently while ensuring robust error handling.

## 🎯 Objective

* Understand Python file handling concepts.
* Read and write data to text and CSV files.
* Automate file operations such as renaming, moving, and deleting files.
* Implement exception handling using `try-except`.
* Write clean, well-commented, and maintainable code.

## ✨ Features

* Read data from a text (`.txt`) file.
* Write data to a text (`.txt`) file.
* Read and write CSV (`.csv`) files.
* Rename files automatically.
* Move files to a different folder.
* Delete files programmatically.
* Handle errors using `try-except`.
* Well-documented source code with comments.

## 🛠️ Technologies Used

* Python 3
* `os` module
* `shutil` module
* `csv` module
