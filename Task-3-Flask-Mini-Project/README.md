# 🌐 Task 3 – Flask Mini Project

## 📌 Overview

This project is a simple web application developed using the Flask framework. It demonstrates the fundamentals of backend web development, including routing, template rendering, form handling, and basic data storage.

## 🎯 Objective

* Build a simple web application using Flask.
* Implement Flask routing and HTML templates.
* Handle user input using GET and POST methods.
* Store submitted data in a text file.
* Create a clean and responsive user interface using Bootstrap.

## ✨ Features

* Flask routing
* HTML templates with Jinja2
* GET and POST form handling
* Basic data storage using a text file
* Responsive interface using Bootstrap 5

## 🛠️ Technologies Used

* Python 3
* Flask
* HTML5
* Bootstrap 5

## ▶️ How to Run

1. Install Flask:

   ```
   pip install flask
   ```
2. Open a terminal in the project folder.
3. Run:

   ```
   python app.py
   ```
4. Open the browser and visit:

   ```
   http://127.0.0.1:5000
   ```
