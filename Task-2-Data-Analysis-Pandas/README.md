# 📊 Task 2 – Data Analysis Using Pandas

## 📌 Overview

This project demonstrates basic data analysis using the **Pandas** library in Python. It includes loading a CSV dataset, inspecting the data, cleaning missing values, filtering records, grouping data, and generating simple insights.

## 🎯 Objective

* Load and inspect a CSV dataset.
* Clean missing or incorrect data.
* Apply filtering, grouping, and aggregation.
* Explain the insights in simple words.

## ✨ Features

* Load data from a CSV file.
* Display dataset information.
* Identify and clean missing values.
* Remove duplicate records.
* Filter data based on conditions.
* Group data by department.
* Calculate average salary using aggregation.
* Generate simple insights from the dataset.

## 🛠️ Technologies Used

* Python 3
* Pandas

## ▶️ How to Run

1. Install Pandas:

   ```
   pip install pandas
   ```
2. Place `data_analysis.py` and `sample_dataset.csv` in the same folder.
3. Run the program:

   ```
   python data_analysis.py
   ```

## 📊 Key Insights

* The dataset was loaded and inspected successfully.
* Missing salary values were identified and handled.
* Duplicate records were removed to improve data quality.
* Employees earning more than 50,000 were filtered.
* The average salary of each department was calculated using grouping and aggregation.
